# marvel-heroes
Marvel heroes SPA

### Installation & Run

#### 1. Clone repository
```sh
git clone https://github.com/aineo/marvel-heroes.git
```

#### 2. Install dependencies
```sh
yarn
```

#### 3. Build
```sh
yarn build
```

#### 4. Run project
```sh
yarn start
```

### Other options

#### Linting
```sh
yarn lint
```

#### Formatting with Prettier
```sh
yarn format
```
